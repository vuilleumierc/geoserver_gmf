geoserver_gmf project
===================

Read the `Documentation <https://camptocamp.github.io/c2cgeoportal/2.7/>`_

Checkout
--------

.. code::

   git clone git@github.com:camptocamp/geoserver_gmf.git

   cd geoserver_gmf

Build
-----

.. code::

  ./build

Run
---

.. code::

   docker-compose up -d
